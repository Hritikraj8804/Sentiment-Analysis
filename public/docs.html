Documentation

Welcome to the Sentiment Analysis Tool documentation. This guide will help you understand and use the tool effectively.

Getting Started

Prerequisites

  • Modern web browser (Chrome, Firefox, Safari, or Edge)
  • CSV files with the following format:
    • Column 1: "Review" (text content)
    • Column 2: "Liked" (sentiment value: 1 for positive, 0 for negative, 2 for neutral)
    • Optional: "Dataset" column to specify the dataset type

Supported Datasets

Restaurant Reviews

Default dataset with restaurant-specific reviews and binary sentiment classification.

Hotel Reviews

Hotel-specific reviews with three-way sentiment classification (positive/negative/neutral).

Movie Reviews

Movie-specific reviews with three-way sentiment classification.

Tech Reviews

Technology product reviews with three-way sentiment classification.

Using the Application

Uploading Data

  1. Click the "Upload CSV" button
  2. Select your CSV file
  3. Choose the dataset type (if not specified in the CSV)
  4. Click "Analyze" to start the analysis

Understanding Results

The analysis provides:

  • Overall sentiment score
  • Sentiment distribution
  • Key phrases and words
  • Confidence scores
  • Dataset-specific insights

Best Practices

CSV File Format

Review,Liked,Dataset "Great experience!",1,"Restaurant" "Poor service.",0,"Hotel" "Average movie.",2,"Movie"

Tips for Best Results

  • Ensure your CSV file follows the required format
  • Use clear, concise text in reviews
  • Include the dataset type when analyzing multiple types of reviews
  • Keep review lengths reasonable (1-2 sentences recommended)

Troubleshooting

Common Issues

File Upload Failed

  • Check file format (must be CSV)
  • Verify file size (max 10MB)
  • Ensure proper column headers

Analysis Errors

  • Verify text encoding (UTF-8 recommended)
  • Check for special characters
  • Ensure proper sentiment values (0, 1, or 2)
